## 第四步：Win7 上用 PuTTY 连接搬瓦工 VPS

Win7 用搬瓦工最常用的 SSH 工具是 **PuTTY**，免费、轻量、不用安装，下载解压直接跑。

配置步骤如下：

1. 打开 PuTTY，在 **Host Name** 填你的 VPS IP 地址
2. **Port** 那栏填 KiwiVM 里看到的 SSH 端口（不是 22）
3. Connection type 选 **SSH**
4. 在 Saved Sessions 里输个名字，点 Save 保存，下次不用再填
5. 点 Open，弹出黑色命令行窗口
6. 第一次连会提示 RSA fingerprint，点 Yes/Accept 继续
7. 输入用户名 `root`，回车
8. 粘贴密码（注意：PuTTY 里密码输入不显示任何字符，直接粘贴回车就好）

连上之后看到 `root@xxx:~#` 就成了。

**小提示**：Win7 上 PuTTY 粘贴是右键点一下，不是 Ctrl+V。第一次不知道这个的人可能会以为密码输错了，其实是没粘进去。

## 用 Xshell 连接也可以

Xshell 比 PuTTY 界面更友好一些，支持多标签、可以直接 Ctrl+V 粘贴，个人版免费使用。

设置方式：

1. 新建会话，协议选 SSH
2. 主机填 IP，端口填搬瓦工 KiwiVM 里显示的 SSH 端口
3. 在认证里填用户名 root 和对应密码
4. 点连接

比 PuTTY 多一个步骤，但多标签管理多台 VPS 的时候方便很多。
